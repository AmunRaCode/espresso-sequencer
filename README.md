# espresso-sequencer

![Architecture diagram](./doc/architecture.svg)

## Development

- Make sure [nix](https://nixos.org/download.html) is installed.
- Activate the enviorment with `nix-shell`, or `nix develop`, or `direnv allow`
  if using [direnv](https://direnv.net/).

## Run the tests

    cargo test

## Building figures

    make doc
